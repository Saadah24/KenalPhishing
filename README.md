#KenalPhishing: Interactive Phishing Awareness System

**KenalPhishing** is a web-based educational platform designed to increase cybersecurity awareness among Malaysians. The system simulates real-world phishing scenarios to help users identify malicious links and social engineering tactics.

## 🚀 Key Features
- **NIST Phish Scale Logic:** Implements progressive difficulty levels to challenge user detection skills.
- **Localized Scenarios:** Tailored content for specific age groups:
  - **Children:** Gaming and social media scams.
  - **Adults:** Banking, LHDN, and Government-related scams.
  - **Elderly:** Authority-based and "love" scams.
- **Immediate Feedback:** Interactive front-end providing instant explanations when a user interacts with a malicious link.

## 🛠️ Tech Stack
- **Backend:** ASP.NET Core
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript
- **Logic Framework:** NIST Phish Scale
